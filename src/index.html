Les memes : de blagues à la norme

Le meme : ce mouvement qui est apparu de nulle part et qui est illustré via des images, vidéos et textes.

Personne n’aurait pensé que le même devienne un phénomène mondiale durant son émergence. Non seulement ce phénomène se propage dans le monde entier, mais il devient également le moyen officiel d’intéractions mondial. Ce n’est plus un simple mot, mais un mode de vie.

Contexte / / Contexte / / Contexte / / Contexte / / Contexte / / Contexte /

Avec la baisse d’attention généralisée, la rapidité de consommation et d’assimilation de contenu et de la tendance en constante évolution, les mêmes ont non seulement monopolisé la place dans la bulle de l’humour, mais s’est aussi vu pratiquer dans différents secteurs dont on ne se serait jamais imaginé.

Une chose est certaine : le même est devenu omniprésent mais surtout, évident.

Memes : Politique / / Memes : Politique / / Memes : Politique / / Memes : Politique / / Memes : Politique / / Memes : Politique /

De tous les domaines existants qui touchent à l’activité humaine, la politique a été le domaine qui a subi les changements les plus radicaux. Les débats durant en moyenne d’une heure et demie, ont été drastiquement raccourcis. Bien que les thématiques abordées soient toujours identiques ou du moins, similaires, l’approche en est totalement différente. L’utilisation des mêmes en ont aussi fait que regarder les débats sont devenus plus divertissants.

Cependant, le même en politique est à double tranchant, il peut tout autant être utiliser dans un but de propagande afin d’aboutir à des fins propres.

Memes : Économie / / Memes : Économie / / Memes : Économie / / Memes : Économie / / Memes : Économie / / Memes : Économie /

“Stonks ou not skonts ?” Ces termes étaient déja utilisé, mais pas dans la bourse, ni dans les banques par exemple. Maintenant, les memes sont utilisés dans le domaine de l’économie. Que ce soit pour faire des blagues sur les actions, les investissements ou même la crypto-monnaie, les memes sont devenus un outil incontournable pour les traders et les investisseurs.

Les memes sont aussi utilisés pour faire exploser la viralité des biens et services : plus un bien ou service est viral, plus il obtient de la valeur, et donc, est plus cher.

Memes : Social / / Memes : Social / / Memes : Social / / Memes : Social / / Memes : Social / / Memes : Social /

Bien que les memes ont été utilisé pour les interactions sociales, l’usage de ceux-ci ont été étendus vers d’autres buts. Depuis la naissance des memes, ceux-ci étaient utilisés pour des raisons humoristiques : ils visaient à faire des blagues. L’utilisation des memes s’est étendus dans la romance, les débats et même l’éducation.
